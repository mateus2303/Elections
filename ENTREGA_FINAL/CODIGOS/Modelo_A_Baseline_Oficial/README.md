# Modelo A — baseline oficial

Este diretório contém a configuração portátil e o comando de execução do Modelo A. O código-fonte compartilhado está em `..\MOTOR_COMUM\src\agregador`.

O Modelo A reproduz a metodologia legada (`legacy-baseline-1.0`) e é a série oficial. Depois da execução, abra:

`..\..\RESULTADOS_ATUAIS\Modelo_A_Baseline_Oficial\apresentacao\Resultados_Modelo_A.xlsx`

O script executa validação antes da publicação. Se houver erro bloqueante, ele interrompe sem substituir a entrega Gold.
