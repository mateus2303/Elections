# Modelo B — candidato experimental

Este diretório contém a configuração portátil e o comando de execução do Modelo B. O código-fonte compartilhado está em `..\MOTOR_COMUM\src\agregador`; as regras adicionais estão em `model_b.py`.

O Modelo B usa meia-vida, janela máxima, teto de concentração por instituto, piso de incerteza e gates mais exigentes. Ele não substitui o Modelo A e não deve ser interpretado como previsão eleitoral.

Depois da execução, abra:

`..\..\RESULTADOS_ATUAIS\Modelo_B_Candidato_Experimental\apresentacao\Resultados_Modelo_B.xlsx`

A planilha contém `Incerteza_Modelo_B` e `Probabilidade_Experimental`. Probabilidades suprimidas permanecem nulas com o motivo registrado.
